سلام!  
این یک لندینگ‌پیج شیک و پویا برای نمایش **فرش‌های دستباف و ماشینی ایرانی** با طراحی مدرن و **RTL** هست.  
هدفش اینه که با انیمیشن‌های جذاب و طراحی کاربرپسند، هنر و فرهنگ غنی ایرانی رو به نمایش بذاره.

---

## ✨ تکنولوژی‌های استفاده‌شده

- ⚛️ **React JS**  
- 🎯 **JavaScript (ES6+)**  
- 🎨 **Tailwind CSS**  
- 🎥 **Framer Motion** (برای انیمیشن‌های نرم و حرفه‌ای)  
- 📦 **React Icons** (برای آیکون‌های شبکه‌های اجتماعی)  

---

## 📱 ویژگی‌ها

- ✅ **طراحی RTL**: کاملاً سازگار با زبان فارسی و چیدمان راست‌به‌چپ  
- 📱 **ریسپانسیو**: طراحی عالی برای موبایل، تبلت و دسکتاپ  
- 🖱️ **انیمیشن‌های تعاملی**: افکت‌های hover (مثل زوم و overlay) و tap برای دکمه‌ها و کارت‌ها  
- 🧩 **ساختار تمیز**: کامپوننت‌های ماژولار (Menu, HeroSlider, Collections, About, CTA, Footer)
