سلام!
این پروژه یه لندینگ‌پیج شیک و حرفه‌ای برای نمایش فرش‌های دستباف و ماشینی ایرانی با طراحی مدرن و RTL هست. هدفش جلب توجه کاربر با انیمیشن‌های روان و طراحی بصری جذابه که حس هنر سنتی رو منتقل کنه.
✨ تکنولوژی‌های استفاده‌شده
⚛️ React JS
🎯 JavaScript (ES6+)
🎨 Tailwind CSS
🎥 Framer Motion (برای انیمیشن‌های نرم و چشم‌نواز)
📦 React Icons (برای آیکون‌های شبکه‌های اجتماعی و منو)
📱 ویژگی‌ها

طراحی RTL: کاملاً مناسب زبان پارسی با چیدمان راست‌به‌چپ.
انیمیشن‌های اسکرول: کامپوننت‌ها (هیرو، کالکشن‌ها، درباره، CTA، فوتر) موقع اسکرول با fade-in و slide-in ظاهر می‌شن.
ریسپانسیو ۱۰۰٪: سازگار با موبایل، تبلت و دسکتاپ.
انیمیشن‌های تعاملی: افکت‌های hover (مثل زوم و overlay) و tap برای دکمه‌ها و کارت‌ها.
کد تمیز: ساختار ماژولار با کامپوننت‌های جدا (Menu, HeroSlider, Collections, About, CTA, Footer).

🚀 نصب و اجرا

پروژه رو کلون کن:
bashgit clone <repository-url>
cd karpet

وابستگی‌ها رو نصب کن:
bashnpm install

پروژه رو اجرا کن:
bashnpm start
مرورگرت رو باز کن و به http://localhost:3000 برو.
